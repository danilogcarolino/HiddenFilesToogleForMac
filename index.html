Hidden Files Toogle for Mac OSX Finder

Switcher hidden files for Mac OSX Finder.

Download the last program executable here: https://github.com/danilogcarolino/HiddenFilesToogleForMac/releases/download/1.0/hiddenfiles

OR

Build the program: gcc -Wall -g main.c -o hiddenfiles

Move the binary to the local bin directory: mv hiddenfiles /usr/local/bin

Use the command - OSX terminal: hiddenfiles on | off or hiddenfiles true | false

Change the execution permission (if necessary): chmod u+x hiddenfiles

The program will only run the appropriate command in background to show or hide your hidden system files.

Enjoy.
